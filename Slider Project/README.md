
📸 Image Slider Project

A simple Image Slider (Carousel) built using HTML, CSS, JavaScript, and Bootstrap.
This project allows users to navigate through images using Previous / Next buttons and also supports auto-sliding every 3 seconds.

- Project live link : https://slider-project-1.netlify.app/

🚀 Features
- 🔁 Next & Previous button navigation
- ⏱️ Auto image sliding (every 3 seconds)
- 🔄 Infinite loop (images repeat)
- 🎨 Clean UI with Bootstrap
- 📱 Responsive layout

🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript 
- Bootstrap 5

📸 How It Works
- Images are stored in an array
- A variable index tracks the current image

  📸Clicking:
    -  → shows next image
    -  → shows previous image
- setInterval() automatically changes image every 3 seconds
