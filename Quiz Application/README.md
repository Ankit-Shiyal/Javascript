# 🧠 Quiz Application

A simple and interactive Quiz Application built using **HTML, CSS, JavaScript, and Bootstrap**.
This project displays multiple-choice questions, tracks user answers, and shows the final result.


## 🚀 Features

* 📋 Multiple-choice quiz questions
* 🎯 Instant answer selection
* 🔢 Question counter (e.g., 1/10)
* ⏭️ Next question functionality
* 🧮 Score calculation
* 📊 Final result display
* 🎨 Responsive UI using Bootstrap


## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Bootstrap 5


## ⚙️ How It Works

1. Quiz loads first question from `quizData`
2. User selects an option
3. Answer is checked automatically
4. Score is updated
5. Next question is displayed
6. At the end, result is shown
